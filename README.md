# 📚 Library Management System

A web-based Library Management System to automate and simplify the daily operations of a library—such as tracking books, managing members, issuing and returning books, and monitoring fines.

---

## 🚀 Features

- 📖 Add, update, and delete books
- 👥 Register and manage members
- 📆 Issue and return books with due-date tracking
- 💸 Calculate and manage fines
- 🔍 Search books and members
- 🔐 Admin authentication system
